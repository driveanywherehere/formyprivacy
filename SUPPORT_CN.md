
# InPrivate Portfolio Tracker - 支持

## 关于

InPrivate Portfolio Tracker 是一款本地优先的 macOS 投资追踪应用。所有数据保存在你的设备上——无需账号，不使用云端，不采集任何数据。

## 常见问题

**问：我的数据存储在哪里？**
答：所有数据通过内嵌数据库存储在本地 Mac 上，不会上传到任何服务器。

**问：如何备份数据？**
答：数据存储在应用的本地目录中，你可以从 `~/Library/Application Support/InPrivate Portfolio Tracker/` 复制数据库文件进行备份。

**问：支持哪些市场？**
答：支持 Yahoo Finance 覆盖的所有市场，包括美股、港股（如 `0700.HK`）、上交所（如 `600519.SS`）、深交所（如 `000858.SZ`）、东京等。

**问：可以追踪哪些资产类型？**
答：股票、期权、期货和现金。

**问：需要联网吗？**
答：仅在获取实时行情时需要联网，其他功能均可离线使用。
